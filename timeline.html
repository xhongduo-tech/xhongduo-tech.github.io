基础奠定 · 1950–2016

// 神经网络的漫长前夜：从哲学命题到工程突破

1950
图灵测试

Alan Turing 在《计算机器与智能》中提出：若机器的回答使人无法分辨其为机器，则视其为"能思考"。AI 哲学问题的起点，而非技术路线图。

1956
Dartmouth 会议

McCarthy、Minsky 等人首次使用"人工智能"一词，确立 AI 为独立研究领域。随后数十年以规则系统、专家系统为主线——路走对了，工具错了。

1986
反向传播算法

Rumelhart、Hinton、Williams 将 BP 算法系统化发表，多层神经网络训练在理论上成立。此后 26 年，算力与数据都不够，理论先等着。

1997
Deep Blue 击败卡斯帕罗夫

IBM 超级计算机在正式比赛中首次击败人类国际象棋世界冠军。象征意义远大于技术意义——这是暴力搜索，不是智能，但全球开始认真对待 AI。

2012
AlexNet — 深度学习革命 架构

Hinton 团队在 ImageNet 上以 top-5 错误率 15.3% 夺冠（第二名 26.2%）。GPU 并行 + 大数据 + 深度 CNN 的组合彻底终结了手工特征工程时代，拉开深度学习序幕。

Transformer 奠基 · 2017–2020

// 统一架构出现，预训练范式确立，规模红利浮现

2017.06
Attention Is All You Need 架构

Google Brain 8 人团队发表 Transformer，纯注意力机制彻底取代 RNN/LSTM。无递归结构使并行训练成为可能，序列建模范式被重写。此后所有大模型的统一底座由此而来。

2018.06
GPT-1 模型

OpenAI 将 Transformer 用于语言模型无监督预训练 + 有监督微调，1.17 亿参数，验证"预训练迁移"路线的可行性。GPT 系列的起点。

2018.10
BERT 架构

Google 提出双向预训练语言模型，MLM（掩码语言模型）+ NSP 任务使上下文理解能力大幅超越单向模型，横扫 11 项 NLP 基准。Encoder-only 路线的集大成者。

2019.02
GPT-2 — "危险到无法发布" 模型

15 亿参数，生成文本质量使 OpenAI 以"滥用风险"为由拒绝完整开放——这是历史上第一次 AI 安全被公开讨论。事后来看，担忧提前了几年，但意识觉醒有价值。

2020.05
GPT-3 — 规模涌现 模型

1750 亿参数，few-shot 能力涌现：无需微调，凭三五示例即可执行翻译、摘要、代码生成。"规模即智能"假说首次获得强证据，Scaling Law 成为主流信条。API 商业化路线同步确立。

2020.12
AlphaFold 2 — AI 进入科学发现 科学

DeepMind 在 CASP14 以近原子精度预测蛋白质三维结构，解决了结构生物学 50 年难题。标志深度学习从感知任务突破至科学发现领域，AI for Science 路线正式确立。

对齐与商业化 · 2021–2022

// RLHF 让模型学会"听话"，生成式 AI 首次大规模落地

2021.06
GitHub Copilot 产品

OpenAI Codex 驱动的代码补全工具进入技术预览，首个在生产环境大规模落地的 LLM 应用。开发者工作流被重构，AI 从研究走向日常工具的第一步。

2022.01
InstructGPT — RLHF 对齐 架构

OpenAI 将人类反馈强化学习（RLHF）引入语言模型，使模型行为从"预测下一 token"转向"遵循人类意图"。规模更小却被用户偏好超过 GPT-3，奠定 ChatGPT 的直接技术基础。

2022.04
DALL·E 2 — 文本生成图像 多模态

扩散模型（Diffusion Model）+ CLIP 使文本到图像质量首次超越公众预期边界。生成式 AI 不再是语言专属，多模态时代的序幕从这里掀开。

2022.08
Stable Diffusion — 开源扩散模型 开源 多模态

Stability AI 开源高质量图像生成模型，将图像生成门槛降至零，消费级 GPU 即可本地运行。LoRA、ControlNet 等社区扩展随即涌现，创作工具生态 Cambrian 爆发。

2022.11.30
ChatGPT — 奇点时刻 产品

GPT-3.5 + RLHF 封装为对话产品，上线 5 天 100 万用户，2 个月突破 1 亿——史上增长最快的消费应用。触发全球 AI 战略重估，微软、谷歌、百度相继进入战时状态。

大爆炸 · 2023

// 闭源与开源两条线同步爆发，Agent 范式首次成型

2023.02
LLaMA — 开源生态引爆 开源

Meta AI 开源 7B–65B 语言模型，泄露版本迅速在社区传播。Alpaca、Vicuna、WizardLM 系列接连涌现，打破闭源垄断，任何人都可以在笔记本上运行并微调大模型。开源 LLM 生态由此奠基。

2023.03
GPT-4 — 多模态旗舰 模型

支持图像输入，跨领域推理大幅提升——律师资格考试成绩超过 90% 人类考生。闭源壁垒与开源生态的对立格局正式确立，"大模型军备竞赛"一词开始频繁出现。

2023.03
Claude 1 — 宪法 AI 模型

Anthropic 发布首版 Claude，采用宪法 AI（Constitutional AI）对齐方法：用原则性规则驱动自我批判与修正，大幅减少人工标注依赖，提供对齐路线的另一种范式。

2023.04
AutoGPT — Agent 范式首次流行 Agent

首个获得广泛关注的 LLM 自主 Agent 框架，将"任务分解—工具调用—自我反思"循环带入公众视野。GitHub 数日内破万星，触发 LangChain、AutoGen、BabyAGI 等 Agent 框架潮。

2023.07
LLaMA 2 — 开源商用 开源

Meta 与微软合作发布，附带商业友好许可证，开源 LLM 首次可合法用于商业产品。Chat 版本同步发布，开源对话模型与 ChatGPT 的差距开始缩小。

2023.09
Mistral 7B — 小模型效率极限 开源

7B 参数超越同规模所有竞品，滑动窗口注意力（SWA）+ 分组查询注意力（GQA）证明架构效率优化空间巨大。随后的 Mixtral-8x7B 将 MoE 路线推向开源主流。

多模态与工具竞速 · 2024

// 感知边界扩展，推理能力开始成为新的分水岭

2024.02
Sora — 视频世界模型 多模态

OpenAI 发布文本生成视频模型，60 秒高保真视频，物理世界连贯性超出预期。"Sora 是世界模型的雏形"的讨论引发学界对视觉预训练路线的重估，但技术细节至今未完全公开。

2024.03
Claude 3 — Opus / Sonnet / Haiku 模型

三档能力/成本定位，Opus 在多项基准首次超越 GPT-4，Anthropic 建立与 OpenAI 对等的竞争地位。分级产品策略成为行业标配。

2024.05
GPT-4o — 全模态实时交互 多模态

端到端全模态架构（文本 / 图像 / 音频），语音响应延迟降至 320ms，实时情感感知，交互体验接近人类对话。"omni"模型重新定义人机交互范式，同步宣布 GPT-4 API 免费化。

2024.06
Claude 3.5 Sonnet — 效率驱动迭代 模型

Claude 系列中Sonnet 档次反超 Opus，以更低成本提供更强能力，验证"效率驱动"模型迭代路线。Computer Use 功能同期公测，Agent 控制计算机的路线正式落地。

2024.09
OpenAI o1 — 推理模型时代开始 推理

显式 chain-of-thought 推理链，用推理时间换精度：数学竞赛、代码调试、科学推断大幅提升，超越博士水平的评测开始出现。"预训练 Scaling"之外，"推理时间 Scaling"成为新主轴。

2024.10
AI IDE 爆发 — Cursor 与编程新范式 产品 Agent

Cursor 完成 $9 亿美元融资，成为增速最快的开发者工具；Windsurf（前 Codeium）、GitHub Copilot Workspace 相继发布。AI 辅助编程从代码补全跨越至上下文感知的协作开发：整个代码库参与推理，PR 级别的任务可"一句话完成"。开发者工作流第二次被彻底重构。

2024.11
Model Context Protocol — Agent 工具标准化 Agent

Anthropic 开源 MCP（模型上下文协议），为 AI 模型访问数据库、文件系统、API 提供统一接口标准。GitHub、Slack、Google Drive 等数十个连接器迅速上线，推动 Agent 生态从各自为战走向互操作。MCP 在 2025 年内成为 AI 开发者工具链的事实基础设施，IDE、框架、平台普遍内置支持。

2024.12
Gemini 2.0 Flash 模型

Google 多模态架构全面升级，实时流式多模态输入输出，Agent 工具调用与 Deep Research 能力增强。Google 经历 2023 年被动局面后，重返 AI 前沿竞争序列。

推理模型竞赛 · 2025 H1

// 开源对齐闭源，思考时间成为核心变量，地缘竞争进入 AI 叙事

2025.01
DeepSeek R1 — 开源推理模型冲击 推理 开源

中国团队发布开源推理模型，性能媲美 OpenAI o1，公开的训练成本仅为数十分之一。冲击"顶级推理能力需要顶级投入"的叙事，引发全球资本市场震荡与地缘 AI 政策重估，英伟达单日市值蒸发近 6000 亿美元。

2025.01
Stargate — 美国 AI 国家基础设施 产品

OpenAI、SoftBank、Oracle 联合宣布 $5000 亿美元（5000 亿）AI 基础设施投资计划，获特朗普政府背书并于白宫发布。AI 从技术领域争论升格为国家战略资产，半导体、算力、能源政策随即联动；AI 数据中心建设进入"战时速度"，地缘 AI 竞争正式进入国家级资源博弈。

2025.01
o3-mini — 可控推理预算 推理

OpenAI 正式发布小型推理模型，首次引入"思考预算"（thinking budget）机制：低 / 中 / 高三档控制推理深度，可在延迟与精度间按需权衡。STEM 基准上超越 o1，成本降低 3–5 倍，将推理能力下沉至高频 API 调用场景，DeepSeek 冲击后加速上线。

2025.02
Claude 3.7 Sonnet — 混合推理 推理

混合推理架构：可控延伸思考深度（标准模式 / 扩展思考模式切换），在软件工程基准 SWE-bench 达到新高点。Anthropic 确立推理能力 + 工程综合能力的双轮路线。

2025.03
GPT-4.5 & o3 模型 推理

GPT-4.5 扩大预训练规模强化世界知识与情感理解；o3 将推理泛化能力推至新基准，在 ARC-AGI 测试中达到接近人类水平。预训练路线与推理路线并行推进，OpenAI 在 DeepSeek 冲击后加速节奏。

2025.04
o3 全面开放 & o4-mini — 推理能力普及化 推理 多模态

o3 向所有 API 用户开放；o4-mini 同步发布，原生支持视觉输入，推理模型首次兼具多模态能力。至此，推理模型从"高端旗舰"演变为可日常集成的基础能力。o1 开启的"推理时间 Scaling"路线，在不到一年内完成从实验到量产的全面转型。

2025.04
LLaMA 4 — 多模态开源旗舰 开源 多模态

Meta 发布 LLaMA 4 系列（Scout 17B / Maverick 17B MoE），原生图像 + 视频 + 文本多模态输入；Scout 支持 1040 万 token 超长上下文，打破开源模型长上下文瓶颈。万亿参数旗舰 Behemoth 同期预告。开源多模态生态正式进入与 GPT-4o 同台竞技的序列。

2025.04
GPT-4.1 — 百万 token 代码专项 模型

OpenAI 发布 GPT-4.1 系列（含 mini / nano），100 万 token 上下文，SWE-bench 代码任务显著提升；GPT-4o 同期宣布再次降价。"功能迭代不停歇 + 旗舰降价提速"成为 OpenAI 在 DeepSeek 冲击后的主要商业竞争节奏。

2025.05
Gemini 2.5 Pro — 超长推理上下文 推理 模型

Google I/O 重磅发布，原生推理能力 + 100 万 token 上下文，代码生成与多步推理基准达到同期最高水平。Google 深度整合 Android / Chrome 生态，推出 AI Overview 全面改造搜索入口，重返 AI 前沿第一梯队——"AI 浏览器战争"由此开始。

2025.05
Qwen 3 — 开源混合推理 开源 推理

阿里巴巴通义团队发布 Qwen 3 系列（0.6B–235B MoE），单模型同时支持"思考模式"与"快速模式"动态切换，无需为推理 vs 非推理场景分别部署。多语言与工具调用能力突出，开源混合推理架构的重要参考实现；DeepSeek 之后中国开源模型再次确立国际竞争力。

Agent 时代开幕 · 2025 H2

// 推理能力普及，Agent 从实验走向生产，AI 编程接管软件开发主流

2025.06
Claude 4 Sonnet — Agent 时代旗舰 模型 Agent

Anthropic 发布 Claude 4 系列，Sonnet 4 在 SWE-bench Verified 达到新高，将代码能力与多步工具调用深度融合。Claude 同步升级 Projects 与 Artifacts 产品功能，从问答工具演变为可持续协作的智能工作台；面向企业的长周期 Agent 部署成为核心场景，Anthropic 确立"安全 + 能力"双轮飞轮的竞争定位。

2025.07
Grok 3 — xAI 入局推理竞赛 推理

马斯克 xAI 发布 Grok 3，内置深度思考模式，整合 X 平台实时数据流实现推理 + 时效的组合定位；Aurora 图像生成能力同期上线。xAI 从单一文本模型扩展为多模态竞争者，AI 赛道的"独立宇宙"格局进一步加剧——五大势力（OpenAI / Anthropic / Google / Meta / xAI）全部拥有顶级模型的局面正式成型。

2025.09
o1-Pro / o3-Pro — 极限推理分层 推理

OpenAI 发布 Pro 档推理模型，面向研究型任务开放分钟级链式思考，博士级科学问题与复杂数学证明基准持续刷新。"前沿推理"与"日常使用"的成本分层愈加明显——o3-mini（低成本快速）/ o3（通用）/ o3-Pro（极限）三档定位形成清晰的推理能力商品货架。

2025.10
AI 编程接管开发主流 产品 Agent

Cursor / Windsurf / GitHub Copilot Workspace 累计付费用户突破千万级，MCP 成为 AI 工具链事实标准。多款 Agent 在 SWE-bench Verified 上突破 50% 解决率：输入 Issue 标题 → Agent 自主完成设计→实现→测试→PR 全流程。"初级工程师"的角色定义开始被严肃讨论，AI-native 软件公司的人效比达到传统公司的数倍。

2025.12
Gemini 2.0 系列收官 & 2025 年度盘点 模型

Google 完整发布 Gemini 2.0 全系列（Ultra / Pro / Flash / Flash-Lite），深度整合 Google Search、Workspace、Android。2025 年度关键结论：推理时间 Scaling 已验证；开源追平闭源旗舰；Agent 从实验进入生产；AI 芯片短缺仍是最大约束；"AGI"的定义争议从未停歇。

AGI 边界试探 · 2026 Q1

2026.03
GPT-5.4 — 推理与多模态新旗舰 模型 推理

OpenAI 发布 GPT-5.4，通用推理与多模态能力统一刷新旗舰基线，将"思考"原生整合（无需切换推理模式）；GPT-5 系列经过数轮迭代后在代码、数学、Agent 综合能力上全面领先。模型本身与 Agent 框架的边界进一步模糊，端到端工作流成为主流交互范式。
