基础奠定 · 1950–2016

// 神经网络的漫长前夜：从哲学命题到工程突破

1950
图灵测试科学

Alan Turing 在《计算机器与智能》中提出：若机器的回答使人无法分辨其为机器，则视其为"能思考"。AI 哲学问题的起点，而非技术路线图。

1956
Dartmouth 会议科学

McCarthy、Minsky 等人首次使用"人工智能"一词，确立 AI 为独立研究领域。随后数十年以规则系统、专家系统为主线——路走对了，工具错了。

1986
反向传播算法架构

Rumelhart、Hinton、Williams 将 BP 算法系统化发表，多层神经网络训练在理论上成立。此后 26 年，算力与数据都不够，理论先等着。

1997
Deep Blue 击败卡斯帕罗夫模型

IBM 超级计算机首次击败人类国际象棋世界冠军。象征意义远大于技术意义——这是暴力搜索，不是智能，但全球开始认真对待 AI。

2012
AlexNet — 深度学习革命 架构

Hinton 团队在 ImageNet 上以 top-5 错误率 15.3% 夺冠（第二名 26.2%）。GPU 并行 + 大数据 + 深度 CNN 的组合彻底终结了手工特征工程时代，拉开深度学习序幕。

2014
GAN — 生成对抗网络 架构

Ian Goodfellow 提出生成器与判别器的博弈训练框架，开创无监督生成建模新范式。图像合成、风格迁移等应用由此奠基；扩散模型普及之前，GAN 是图像生成主流路线，也是深度学习从「识别」走向「创造」的标志性一步。

2016.03
AlphaGo 击败李世石 — 深度强化学习的里程碑 模型

DeepMind AlphaGo 以 4:1 击败人类围棋世界冠军。深度强化学习 + 蒙特卡罗树搜索的组合改写了预期边界，全球对 AI 能力的认知被重置。「AI 能在复杂策略游戏中超越人类专家」首次得到验证，为 AlphaZero、AlphaFold 系列奠定方法论基础。

Transformer 奠基 · 2017–2020

// 统一架构出现，预训练范式确立，规模红利浮现

2017.06
Attention Is All You Need 架构

Google Brain 8 人团队发表 Transformer，纯注意力机制彻底取代 RNN/LSTM。无递归结构使并行训练成为可能，序列建模范式被重写。此后所有大模型的统一底座由此而来。

2018.06
GPT-1 模型

OpenAI 将 Transformer 用于无监督预训练 + 有监督微调，验证"预训练迁移"路线的可行性。GPT 系列的起点。

2018.10
BERT 架构

Google 提出双向预训练语言模型，MLM + NSP 任务使上下文理解能力大幅超越单向模型，横扫 11 项 NLP 基准。Encoder-only 路线的集大成者。

2019.02
GPT-2 — "危险到无法发布" 模型

15 亿参数，生成质量使 OpenAI 以"滥用风险"为由拒绝完整开放。这是历史上第一次 AI 安全被公开讨论，意识觉醒比时间表早了几年。

2020.05
GPT-3 — 规模涌现 模型

1750 亿参数，few-shot 能力涌现：无需微调，凭三五示例即可执行翻译、摘要、代码生成。"规模即智能"假说首次获得强证据，Scaling Law 成为主流信条，API 商业化路线同步确立。

2020.12
AlphaFold 2 — AI 进入科学发现 科学

DeepMind 在 CASP14 以近原子精度预测蛋白质三维结构，解决了结构生物学 50 年难题。标志深度学习从感知任务突破至科学发现领域，AI for Science 路线正式确立。

对齐与商业化 · 2021–2022

// RLHF 让模型学会"听话"，生成式 AI 首次大规模落地

2021.06
GitHub Copilot 产品

OpenAI Codex 驱动的代码补全工具进入技术预览，首个在生产环境大规模落地的 LLM 应用。开发者工作流被重构，AI 从研究走向日常工具的第一步。

2022.01
InstructGPT — RLHF 对齐 架构

OpenAI 将人类反馈强化学习（RLHF）引入语言模型，使模型行为从"预测下一 token"转向"遵循人类意图"。规模更小却被用户偏好超过 GPT-3，奠定 ChatGPT 的直接技术基础。

2022.04
DALL·E 2 — 文本生成图像 多模态

扩散模型（Diffusion Model）+ CLIP 使文本到图像质量首次超越公众预期边界。生成式 AI 不再是语言专属，多模态时代的序幕从这里掀开。

2022.08
Stable Diffusion — 开源扩散模型 开源 多模态

Stability AI 开源高质量图像生成模型，将图像生成门槛降至零，消费级 GPU 即可本地运行。LoRA、ControlNet 等社区扩展随即涌现，创作工具生态 Cambrian 爆发。

2022.11.30
ChatGPT — 奇点时刻 产品

GPT-3.5 + RLHF 封装为对话产品，上线 5 天 100 万用户，2 个月突破 1 亿——史上增长最快的消费应用。触发全球 AI 战略重估，微软、谷歌、百度相继进入战时状态。

大爆炸 · 2023

// 闭源与开源两条线同步爆发，搜索巨头被迫应战，Agent 范式首次成型

2023.02

微软将 GPT-4 提前集成进 Bing，把 LLM 能力带入十亿级搜索入口。谷歌随即紧急宣布 Bard，触发全球搜索格局重估。微软百亿美元 OpenAI 投资的战略意图首次完整呈现，"AI 搜索"作为独立叙事正式登场。

2023.02

Meta AI 开源 7B–65B 语言模型，泄露版本迅速在社区传播。Alpaca、Vicuna、WizardLM 系列接连涌现，打破闭源垄断，任何人都可以在笔记本上运行并微调大模型。开源 LLM 生态由此奠基。

2023.03
GPT-4 — 多模态旗舰 模型

支持图像输入，跨领域推理大幅提升——律师资格考试成绩超过 90% 人类考生。闭源壁垒与开源生态的对立格局正式确立，"大模型军备竞赛"一词开始频繁出现。

2023.03

Anthropic 发布首版 Claude，采用宪法 AI（Constitutional AI）对齐方法：用原则性规则驱动自我批判与修正，大幅减少人工标注依赖，提供对齐路线的另一种范式。

2023.04
AutoGPT — Agent 范式首次流行 Agent

首个广泛传播的 LLM 自主 Agent 框架，将"任务分解—工具调用—自我反思"循环带入公众视野。GitHub 数日内破万星，触发 LangChain、BabyAGI 等框架潮。但实际落地能力有限，更多意义在于普及了 Agent 范式的概念。

Anthropic 将上下文窗口扩展至 10 万 token（可容纳整本小说），长上下文竞赛由此成为各大模型核心差异化方向，RAG 架构设计因此被重新评估。

2023.07

Meta 与微软合作发布，附带商业友好许可证，开源 LLM 首次可合法用于商业产品，开源对话模型与 ChatGPT 的差距开始缩小。

2023.09

首个获广泛部署的开源混合专家（MoE）模型：8 个专家，每次激活 2 个，以约 12B 活跃参数实现 70B 级别效果。MoE 路线从论文走入开源生产，成为后续 DeepSeek、LLaMA 4 等模型的标准选项。
